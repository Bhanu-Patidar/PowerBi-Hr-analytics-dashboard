# HR Analytics Dashboard – Power BI

## 📌 Project Objective
Analyze employee attrition and workforce trends using interactive reporting.

## 🛠 Tools Used
- Power BI
- DAX
- Data Modeling

## 📊 Key Features
- Attrition rate tracking
- Department-wise analysis
- KPI metrics
- Interactive filters and slicers

## 📈 Insights Generated
- Identified departments with higher attrition
- Analyzed workforce distribution trends
- Built dashboard to support HR decision-making
